二、项目流程
大家看文件结构，在evals.py里面，现在实现了所有的5个评估函数的框架，但是还没有开始具体调参。我现在设想的项目流程是：对于每一个评估函数，都要找到一个最好的参数（参数是feature的权重），具体可能要写一下脚本。这五个评估函数找到各自最好的参数后，各跑100组实验，最后给出一些评估指标，来对比这些函数的优点和缺点（比如单步决策速度，最后成功率，以及成功时耗费的时间或者步数等）。
